# Reset-OrchProcessVersion

## SYNOPSIS
Reverts the package version of the processes to the last version.

## SYNTAX

```
Reset-OrchProcessVersion [-Name] <String[]> [-Path <String[]>] [-Recurse] [-Depth <UInt32>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```
