# Get-OrchUpdateSetting

## SYNOPSIS
Gets the update settings from UiPath Orchestrator.

## SYNTAX

```
Get-OrchUpdateSetting [-Path <String[]>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
The `Get-OrchUpdateSetting` cmdlet retrieves the update configuration settings from UiPath Orchestrator. This includes the update server source, update server URL, and polling interval settings that control how Orchestrator checks for and manages updates.

These settings determine how the Orchestrator instance connects to update servers and how frequently it polls for available updates.

Multiple values for the -Path parameter can be specified using comma-separated text that includes wildcards. Additionally, you can use autocomplete for these values by pressing [Ctrl+Space] or [Tab].

Primary Endpoint: GET /odata/Settings/UiPath.Server.Configuration.OData.GetUpdateSettings

OAuth required scopes: OR.Settings or OR.Settings.Read

Required permissions: [PLACEHOLDER - Update settings view permissions]

## EXAMPLES

### Example 1
```powershell
PS C:\> Set-Location Orch1:
PS Orch1:\> Get-OrchUpdateSetting
```

Gets the update settings for the current Orchestrator instance.

### Example 2
```powershell
PS Orch1:\> $updateSetting = Get-OrchUpdateSetting
PS Orch1:\> Write-Host "Update server: $($updateSetting.UpdateServerUrl)"
PS Orch1:\> Write-Host "Polling interval: $($updateSetting.PollingInterval) minutes"
```

Retrieves the update settings and displays the update server URL and polling interval.

### Example 3
```powershell
PS Orch1:\> Get-OrchUpdateSetting | Select-Object UpdateServerSource, UpdateServerUrl, PollingInterval
```

Gets the update settings and displays only the key configuration properties.

### Example 4
```powershell
PS Orch1:\> $setting = Get-OrchUpdateSetting
PS Orch1:\> if ($setting.UpdateServerSource -eq "Orchestrator") {
>>     Write-Host "Using Orchestrator as update source"
>> } else {
>>     Write-Host "Using custom update source: $($setting.UpdateServerSource)"
>> }
```

Checks the update server source and displays appropriate information.

## NOTES
- The UpdateServerSource indicates the type of update source (e.g., "Orchestrator")
- The UpdateServerUrl contains the full URL of the update server
- The PollingInterval specifies how often (in minutes) the system checks for updates
- These settings control the automatic update checking behavior of the Orchestrator instance
