# Get-OrchUpdateSetting

## SYNOPSIS
Gets the update settings from UiPath Orchestrator.

## SYNTAX

```
Get-OrchUpdateSetting [-Path <String[]>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
The Get-OrchUpdateSetting cmdlet retrieves the update configuration settings from UiPath Orchestrator. This includes the update server source, update server URL, and polling interval settings that control how Orchestrator checks for and manages updates.

This is a tenant-scoped entity. The -Path parameter specifies the drive name (e.g., Orch1:, Orch2:) to target specific Orchestrator tenants.

Primary Endpoint: GET /odata/Settings/UiPath.Server.Configuration.OData.GetUpdateSettings

OAuth required scopes: OR.Settings or OR.Settings.Read

Required permissions: Settings.View

## EXAMPLES

### Example 1: Get update settings for current tenant
```powershell
PS Orch1:\> Get-OrchUpdateSetting -Path Orch1:
```

Gets the update settings for the Orch1 tenant.

### Example 2: Get update settings from multiple tenants
```powershell
PS C:\> Get-OrchUpdateSetting -Path Orch1:, Orch2:, Orch3:
```

Gets the update settings from multiple Orchestrator tenants.

### Example 3: Display specific properties
```powershell
PS C:\> Get-OrchUpdateSetting -Path Orch1: | Select-Object Path, UpdateServerSource, PollingInterval
```

Gets the update settings and displays only the key configuration properties.

### Example 4: Convert to JSON for detailed analysis
```powershell
PS C:\> Get-OrchUpdateSetting -Path Orch1:, Orch2: | ConvertTo-Json
```

Gets update settings from multiple tenants and converts to JSON for detailed analysis.

## NOTES
- UpdateServerSource indicates the type of update source (typically "Orchestrator")
- UpdateServerUrl contains the full URL of the update server
- PollingInterval specifies the frequency (in minutes) for checking updates
- These settings control the automatic update checking behavior
