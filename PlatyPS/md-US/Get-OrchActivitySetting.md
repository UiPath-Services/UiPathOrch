# Get-OrchActivitySetting

## SYNOPSIS
Gets the activity settings from UiPath Orchestrator.

## SYNTAX

```
Get-OrchActivitySetting [-Path <String[]>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
The `Get-OrchActivitySetting` cmdlet retrieves the activity-related configuration settings from UiPath Orchestrator. This includes the API version used for activities, SignalR connection settings for real-time communication with activities, and related configuration options.

These settings control how activities communicate with Orchestrator and manage real-time signaling for activity execution and monitoring.

Multiple values for the -Path parameter can be specified using comma-separated text that includes wildcards. Additionally, you can use autocomplete for these values by pressing [Ctrl+Space] or [Tab].

Primary Endpoint: GET /odata/Settings/UiPath.Server.Configuration.OData.GetActivitySettings

OAuth required scopes: OR.Settings or OR.Settings.Read

Required permissions: [PLACEHOLDER - Activity settings view permissions]

## EXAMPLES

### Example 1
```powershell
PS Orch1:\> Get-OrchActivitySetting
```

Gets the activity settings for the current Orchestrator instance.

### Example 2
```powershell
PS Orch1:\> $setting = Get-OrchActivitySetting; $setting.SignalR
```

Retrieves the activity settings and displays the SignalR configuration details.

### Example 3
```powershell
PS C:\> Get-OrchActivitySetting -Path Orch1:, Orch2:
```

Gets activity settings from multiple tenants.

### Example 4
```powershell
PS Orch1:\> Get-OrchActivitySetting | ConvertTo-Json -Depth 2
```

Gets activity settings and displays the complete structure including nested SignalR properties.

## NOTES
- The ApiVersion property indicates the API version used for activity communication
- The SignalR.Url property contains the URL for real-time SignalR communication
- The SignalR.SkipNegotiation property controls whether SignalR negotiation is bypassed
- These settings are essential for proper activity execution and real-time monitoring
