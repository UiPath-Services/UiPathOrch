# Disable-OrchUserAttended

## SYNOPSIS
Disables the users to run automations.

## SYNTAX

```
Disable-OrchUserAttended [-UserName] <String[]> [-Path <String[]>] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The MayHaveRobotSession property of the specified users will be set to False.

Primary Endpoint: PUT /odata/Users({userId})

OAuth required scopes: OR.Users

Required permissions: Users.Edit or Robots.Create or Robots.Edit or Robots.Delete
