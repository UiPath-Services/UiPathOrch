# Remove-OrchQueue

## SYNOPSIS
Removes the queues.

## SYNTAX

```
Remove-OrchQueue [-Name] <String[]> [-Path <String[]>] [-Recurse] [-Depth <UInt32>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Remove-OrchQueue cmdlet permanently deletes queues from UiPath Orchestrator. This operation removes the queue definition along with any associated configuration, but does not affect queue items that have already been processed or are currently in the queue.

This is a folder entity cmdlet. Use Set-Location cmdlet (cd command) to navigate to the target folder first, or specify the target folders using -Path, -Recurse, or -Depth parameters. The -Recurse parameter enables removing queues from all subfolders.

The cmdlet supports wildcard patterns for queue names, enabling bulk deletion of multiple queues that match specific criteria. Use the -WhatIf parameter to preview which queues would be removed before executing the operation.

WARNING: Queue removal is a destructive operation and cannot be undone. Ensure you have appropriate backups and that the queues are no longer needed before deletion. Active queue items may be lost during the removal process.

The cmdlet includes built-in confirmation prompts for safety. Use -Confirm:$false to bypass confirmation prompts for automated scripts, but exercise extreme caution when doing so.

Primary Endpoint: GET /odata/QueueDefinitions, DELETE /odata/QueueDefinitions({queueId})

OAuth required scopes: OR.Queues or OR.Queues.Write

Required permissions: Queues.Delete

## EXAMPLES

### Example 1
```powershell
PS Orch1:\Shared> Remove-OrchQueue TestQueue -WhatIf
```

Tests removing a queue using -WhatIf to preview the operation before execution without actually deleting the queue.

### Example 2
```powershell
PS Orch1:\Shared> Remove-OrchQueue TestQueue -Confirm
```

Removes a queue with explicit confirmation prompt for safety verification.

### Example 3
```powershell
PS Orch1:\Shared> Remove-OrchQueue OldQueue001, OldQueue002, OldQueue003
```

Removes multiple specific queues in a single operation by providing a list of queue names.

### Example 4
```powershell
PS Orch1:\Shared> Remove-OrchQueue Test* -WhatIf
```

Tests removing all queues with names starting with Test using wildcard pattern matching.

### Example 5
```powershell
PS Orch1:\> Remove-OrchQueue "Orch1:\Development","Orch1:\Testing" -Name "*Temp*" -Recurse -WhatIf
```

Tests removing all queues containing Temp in their names across Development and Testing folders and their subfolders.

### Example 6
```powershell
PS Orch1:\Shared> Get-OrchQueue | Where-Object {$_.CreationTime -lt (Get-Date).AddMonths(-6)} | Remove-OrchQueue -WhatIf
```

Tests removing queues older than 6 months using pipeline input from Get-OrchQueue with filtering.

### Example 7
```powershell
PS C:\> Remove-OrchQueue Orch1: -Name LegacyQueue -Confirm:$false
```

Removes a specific queue from the tenant root without confirmation prompts, useful for automated cleanup scripts.

### Example 8
```powershell
PS Orch1:\Shared> Remove-OrchQueue Archive* -Depth 2 -WhatIf
```

Tests removing all queues starting with Archive within 2 folder levels deep from the current location.
