# Export-OrchBucketItem

## SYNOPSIS
Exports bucket items from UiPath Orchestrator storage buckets to the local file system.

## SYNTAX

```
Export-OrchBucketItem [[-Name] <String[]>] [[-FullPath] <String[]>] [[-Destination] <String>] [-Path <String>]
 [-Recurse] [-Depth <UInt32>] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Export-OrchBucketItem cmdlet exports files and folders from UiPath Orchestrator storage buckets to the local file system. You can export specific items by name, full path, or export all items from buckets. The cmdlet preserves the folder structure during export and supports recursive operations across multiple folders and buckets.

Primary Endpoint: GET /odata/Buckets({bucketId})/UiPath.Server.Configuration.OData.GetReadUri?path={fullPath}

OAuth required scopes: OR.Administration

Required permissions: Buckets.View and BlobFiles.View

## EXAMPLES

### Example 1
```powershell
PS Orch1:\Shared> Export-OrchBucketItem
```

Exports all files from all buckets in the current folder to the current directory on the local drive.

### Example 2
```powershell
PS Orch1:\Shared> Export-OrchBucketItem MyBucket
```

Exports all files from the TestBucket bucket in the current folder to the current directory on the local drive.

### Example 3
```powershell
PS Orch1:\Shared> Export-OrchBucketItem * Book1.csv, Untitled.txt c:\tmp\backup
```

Exports the specified files from all buckets in the current folder to the specified local directory.

### Example 4
```powershell
PS C:\> Export-OrchBucketItem -Path Orch1:\ -Recurse -Destination c:\tmp\backup
```

Exports all files from all buckets in the Shared folder and its subfolders to the specified local directory.

### Example 5
```powershell
PS Orch1:\Shared> Export-OrchBucketItem * *.csv -WhatIf
```

Shows what CSV files would be exported from all buckets in the Shared folder without actually performing the export.
