# Get-OrchPackage

## SYNOPSIS
プロセスパッケージを取得します。

## SYNTAX

```
Get-OrchPackage [[-Id] <String[]>] [-Path <String[]>] [-Recurse] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
{{ Fill in the Description }}

主に呼び出すエンドポイント: GET /odata/Processes&feedId={feedId}

OAuth に必要なスコープ: OR.Execution or OR.Execution.Read

必要な権限: (Packages.View - Lists packages in a Tenant Feed) and (FolderPackages.View - Lists packages in a Folder Feed)
