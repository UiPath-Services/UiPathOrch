# Remove-OrchRoleFromUser

## SYNOPSIS
ユーザーからロールを外します。

## SYNTAX

```
Remove-OrchRoleFromUser [[-UserName] <String[]>] [-FullName <String[]>] [-Roles] <String[]> [-Path <String[]>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```
