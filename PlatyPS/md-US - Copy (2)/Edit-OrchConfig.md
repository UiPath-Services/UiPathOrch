# Edit-OrchConfig

## SYNOPSIS
Opens the UiPath Orchestrator configuration file for editing.

## SYNTAX

```
Edit-OrchConfig [-EditorType <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
The Edit-OrchConfig cmdlet opens the UiPathOrchConfig.json configuration file in the default or specified editor. This file contains the connection configurations for all UiPath Orchestrator drives, including URLs, authentication settings, and other connection parameters.

The configuration file is located at:
([Environment]::GetFolderPath('MyDocuments'))\PowerShell\Modules\UiPathOrch\UiPathOrchConfig.json

Primary Endpoint: N/A (Local file operation)

OAuth required scopes: N/A

Required permissions: File system write access to user documents folder

## EXAMPLES

### Example 1: Open configuration file in default editor
```powershell
PS C:\> Edit-OrchConfig
```

Opens the UiPathOrchConfig.json file in the default text editor (typically Notepad on Windows).

### Example 2: Initial setup workflow
```powershell
PS C:\> # Check if drives are already configured
PS C:\> Get-OrchPSDrive
PS C:\> # If no drives found, edit configuration
PS C:\> if ((Get-OrchPSDrive).Count -eq 0) {
    Write-Host "No drives configured. Opening configuration editor..."
    Edit-OrchConfig
    Write-Host "After saving changes, restart PowerShell and import modules:"
    Write-Host "Import-Module UiPathOrch, PSReadLine"
}
```

Shows the typical initial setup workflow when no drives are configured.

### Example 3: Open with specific editor
```powershell
PS C:\> Edit-OrchConfig Notepad
```

Opens the configuration file using Windows Notepad.

### Example 4: Open with default JSON editor
```powershell
PS C:\> Edit-OrchConfig Default
```

Opens the configuration file using the default editor associated with .json files (e.g., Visual Studio Code, Notepad++, or other JSON editors).

## NOTES
After modifying the configuration file:
1. Save the file in the editor
2. Restart PowerShell session
3. Import required modules: Import-Module UiPathOrch, PSReadLine
4. Verify configuration with Get-OrchPSDrive
5. Test connection with Get-OrchCurrentUser

The configuration file contains sensitive information including OAuth settings. Ensure proper file permissions and avoid sharing this file.
