# Cmdlet Reference

UiPathOrch exports cmdlets across four API surfaces:

| Prefix | API |
|---|---|
| `Orch` | Orchestrator OData API |
| `Pm`   | Platform Management / Identity Server API |
| `Du`   | Document Understanding API |
| `Tm`   | Test Manager API |

Use the **search box** at the top of the page to jump straight to a cmdlet, or browse the alphabetical list in the sidebar.

For a categorized walkthrough grouped by topic (User Management, Job Management, Triggers, etc.), see [Cmdlets by Topic]({{ '/cmdlets-by-topic/' | relative_url }}).

From a PowerShell session:

```powershell
Get-Help <CmdletName> -Examples   # inline examples
Get-Help <CmdletName> -Online     # open this site for the cmdlet
```
